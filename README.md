**🏫 浦城县第一中学**  
始建于1921年，位于闽浙赣三省交界的福建省浦城县，是福建省一级达标高中。学校设皇华与育贤双校区，其中育贤校区于2022年启用实行全封闭寄宿制，秉承“以善先人、因材施教、博观深悟、强学力行”的校训精神，连续九届获评省级文明校园。

<a name="校规校纪"></a>
## 📜 校规校纪

### 重要规定速览
1. **考勤制度**：缺课超过1/3学时取消考试资格
2. **宿舍管理**：禁止使用大功率电器（>800W）
3. **学术诚信**：抄袭/作弊将受纪律处分
4. **校园安全**：22:00后离校需登记

> ⚠️ 违规后果：  
> - 第一次：警告处分  
> - 第二次：通报批评  
> - 第三次：取消评优资格  

[校规全文 >](regulations/README.md)

### 🚀 如何参与贡献
1. 在GitHub上Fork本项目
2. 克隆你的仓库到本地：
   ```bash
   git clone https://github.com/你的用户名/school-encyclopedia.git
   ```
3. 创建新分支：
   ```bash
   git checkout -b feature/新内容
   ```
4. 添加或修改内容
5. 提交变更：
   ```bash
   git add .
   git commit -m "添加了XX信息"
   ```
6. 推送到你的仓库：
   ```bash
   git push origin feature/新内容
   ```
